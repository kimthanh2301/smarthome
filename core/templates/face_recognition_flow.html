Chi tiết Thuật toán

Algorithm Details
1
Khởi tạo Face Analyzer
Tải model InsightFace buffalo_l với các provider theo thứ tự ưu tiên: CUDA → DirectML → CPU
from insightface.app import FaceAnalysis face_analyzer = FaceAnalysis(name='buffalo_l') face_analyzer.prepare(ctx_id=0, det_size=(640, 640))
2
Load Face Database
Đọc tất cả FaceUser đang active từ database, tính toán embedding trung bình từ nhiều ảnh của mỗi người
# Tính trung bình embedding cho mỗi người averaged_embedding = np.mean(embeddings, axis=0)
3
Phát hiện & Trích xuất
Với mỗi frame từ camera, phát hiện khuôn mặt và trích xuất vector embedding 512 chiều
faces = analyzer.get(frame) for face in faces: embedding = face.normed_embedding # 512D vector
4
So sánh Cosine Similarity
So sánh embedding của khuôn mặt phát hiện được với tất cả embedding trong database
from sklearn.metrics.pairwise import cosine_similarity similarity = cosine_similarity([face_emb], [db_emb])[0][0] if similarity > 0.5: # threshold # Nhận diện thành công!
5
Ghi log & Điều khiển
Nếu nhận diện thành công và đủ khoảng cách thời gian (15s), ghi log điểm danh và gửi lệnh mở cửa qua MQTT
if can_record_attendance(user_code): log_attendance(face_user, 'check', confidence) send_door_open_command(face_user) # MQTT
